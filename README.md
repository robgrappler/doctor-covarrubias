# Sitio Dr. Alexis Covarrubias — Urología · Monterrey

Sitio web de una sola página para el Dr. Alexis Eduardo Covarrubias Benítez, urólogo en Monterrey, N.L.

`index.html` es un archivo autónomo: estilos en línea, fotografías incrustadas y sin JavaScript. Se abre en cualquier navegador (incluido iPhone) sin dependencias ni pasos de compilación.

## Ver localmente
Abre `index.html` en tu navegador, o sirve la carpeta:

```
python3 -m http.server
```

## Publicar con GitHub Pages
1. Sube este repositorio a GitHub.
2. En **Settings → Pages**, elige la rama `main` y la carpeta raíz (`/`).
3. El sitio quedará disponible en `https://<usuario>.github.io/<repo>/`.
